🎯 How to Query with GRO API (For Humans & Agents)

The GRO RESTful API provides 5-dimensional navigation for the entire GRO data universe. All catalog metadata lives in properties.gro_metadata.

🔌 API Base URL

http://localhost:9000/api/v1

Five Dimensions

Collections · 21
Formats · 56
Status · 12
Ownership · 32
Geography · 4 levels

💡 Example Queries

# List all collections
GET /api/v1/collections

# Get all AI agents
GET /api/v1/collections/ai_agent

# List all continents
GET /api/v1/geography/continents

# Unified search with filters
GET /api/v1/resources?collection=ai_agent&status=active

💡 Quick Tips

  • Use 🔍 Unified Search to combine multiple filters across all dimensions
  • Use 📚 Collections to browse by resource type (AI agents, databases, etc.)
  • Use 📦 Formats to find resources by data format (CSV, API, Lambda, etc.)
  • Use 🚦 Status to filter by deployment state (active, draft, implemented)
  • Use 👥 Owner to see what each team owns
  • Use 🌍 Geography to navigate the 4-level hierarchy (continent → country → state → city)
  • Check the API Docs for complete endpoint reference
